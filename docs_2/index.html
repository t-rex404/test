PowerShell _lib ライブラリ

WebDriver、Office Driver、エラー管理などの包括的なPowerShellライブラリ

ライブラリ概要

このライブラリは、PowerShellでのWeb自動化、Office文書操作、エラー管理を統合的に提供する包括的なソリューションです。

主な機能

  • Web自動化: Chrome、Edgeブラウザの自動操作
  • Office文書操作: Word、Excel、PowerPoint文書の作成、編集、保存
  • エラー管理: 包括的なエラーハンドリングとログ機能
  • 共通機能: 再利用可能なユーティリティ関数

基本的な使用方法

1. ライブラリのインポート

# 必要なライブラリをインポート
. "$PSScriptRoot\_lib\WebDriver.ps1"
. "$PSScriptRoot\_lib\ChromeDriver.ps1"
. "$PSScriptRoot\_lib\EdgeDriver.ps1"
. "$PSScriptRoot\_lib\WordDriver.ps1"
. "$PSScriptRoot\_lib\ExcelDriver.ps1"
. "$PSScriptRoot\_lib\PowerPointDriver.ps1"
. "$PSScriptRoot\_lib\Common.ps1"

2. ChromeDriverの使用例

# ChromeDriverの初期化
$chromeDriver = [ChromeDriver]::new()

# ページに移動
$chromeDriver.NavigateToUrl("https://example.com")

# 要素を検索して操作
$element = $chromeDriver.FindElementByCssSelector("#search-input")
$chromeDriver.SetElementText($element, "検索キーワード")

# ブラウザを閉じる
$chromeDriver.Dispose()

3. WordDriverの使用例

# WordDriverの初期化
$wordDriver = [WordDriver]::new()

# テキストを追加
$wordDriver.AddText("Hello, World!")

# 文書を保存
$wordDriver.SaveDocument("C:\output\document.docx")

# WordDriverを閉じる
$wordDriver.Dispose()

4. ExcelDriverの使用例

# ExcelDriverの初期化
$excelDriver = [ExcelDriver]::new()

# セルに値を設定
$excelDriver.SetCellValue("A1", "Hello, Excel!")

# ワークブックを保存
$excelDriver.SaveWorkbook("C:\output\workbook.xlsx")

# ExcelDriverを閉じる
$excelDriver.Dispose()

5. PowerPointDriverの使用例

# PowerPointDriverの初期化
$powerPointDriver = [PowerPointDriver]::new()

# スライドを追加
$powerPointDriver.AddSlide()

# タイトルを設定
$powerPointDriver.SetTitle("プレゼンテーションタイトル")

# プレゼンテーションを保存
$powerPointDriver.SavePresentation("C:\output\presentation.pptx")

# PowerPointDriverを閉じる
$powerPointDriver.Dispose()

注意事項

  • 各ドライバーは必ずDispose()メソッドでリソースを解放してください
  • エラーハンドリングを適切に実装してください
  • ブラウザドライバーを使用する際は、対象のブラウザがインストールされていることを確認してください
  • Office Driverを使用する際は、Microsoft Officeがインストールされていることを確認してください
  • すべてのエラーログは統一されたファイル.\AllDrivers_Error.logに出力されます

エラーコード体系

このライブラリでは、体系的なエラーコードを使用してエラーを管理しています。すべてのエラーは統一されたログファイルに出力されます。

エラーコード範囲 カテゴリ 説明
1001-1130 WebDriver関連 WebDriverクラスの全機能に関するエラー（初期化、ナビゲーション、要素操作、ウィンドウ操作、スクリーンショット、JavaScript実行、ストレージ操作）
2001-2005 EdgeDriver関連 EdgeDriverクラス専用のエラー（初期化、実行ファイル検出、ユーザーデータディレクトリ、デバッグモード、Dispose）
3001-3005 ChromeDriver関連 ChromeDriverクラス専用のエラー（初期化、実行ファイル検出、ユーザーデータディレクトリ、デバッグモード、Dispose）
4001-4016 WordDriver関連 WordDriverクラス専用のエラー（初期化、テキスト操作、コンテンツ操作、ファイル操作、フォーマット、リソース管理）
5001-5016 ExcelDriver関連 ExcelDriverクラス専用のエラー（初期化、セル操作、ワークシート管理、ファイル操作、フォーマット、リソース管理）
6001-6016 PowerPointDriver関連 PowerPointDriverクラス専用のエラー（初期化、スライド操作、コンテンツ追加、ファイル操作、デザイン、リソース管理）
