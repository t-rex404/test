PowerShell _lib ライブラリ

WebDriver、WordDriver、エラー管理などの包括的なPowerShellライブラリ

ライブラリ概要

このライブラリは、PowerShellでのWeb自動化、Word文書操作、エラー管理を統合的に提供する包括的なソリューションです。

主な機能

  • Web自動化: Chrome、Edgeブラウザの自動操作
  • Word文書操作: Word文書の作成、編集、保存
  • エラー管理: 包括的なエラーハンドリングとログ機能
  • 共通機能: 再利用可能なユーティリティ関数

基本的な使用方法

1. ライブラリのインポート

# 必要なライブラリをインポート
. "$PSScriptRoot\_lib\WebDriver.ps1"
. "$PSScriptRoot\_lib\ChromeDriver.ps1"
. "$PSScriptRoot\_lib\WordDriver.ps1"
. "$PSScriptRoot\_lib\Common.ps1"

2. ChromeDriverの使用例

# ChromeDriverの初期化
$chromeDriver = [ChromeDriver]::new()

# ページに移動
$chromeDriver.NavigateToUrl("https://example.com")

# 要素を検索して操作
$element = $chromeDriver.FindElementByCssSelector("#search-input")
$chromeDriver.SetElementText($element, "検索キーワード")

# ブラウザを閉じる
$chromeDriver.Dispose()

3. WordDriverの使用例

# WordDriverの初期化
$wordDriver = [WordDriver]::new()

# テキストを追加
$wordDriver.AddText("Hello, World!")

# 文書を保存
$wordDriver.SaveDocument("C:\output\document.docx")

# WordDriverを閉じる
$wordDriver.Dispose()

注意事項

  • 各ドライバーは必ずDispose()メソッドでリソースを解放してください
  • エラーハンドリングを適切に実装してください
  • ブラウザドライバーを使用する際は、対象のブラウザがインストールされていることを確認してください
  • WordDriverを使用する際は、Microsoft Wordがインストールされていることを確認してください
