⚠️ 重要な注意事項

🔐 セキュリティについて

  • パスワードやトークンは安全に管理してください
  • ブラウザ操作時は個人情報の取り扱いに注意
  • 本番環境での使用前に十分なテストを実施

💡 使用前の準備

  • 必要なアプリケーション（Chrome、Office等）のインストール
  • PowerShell 5.1以上の環境が必要
  • 管理者権限での実行が必要な場合があります

✅ ベストプラクティス

  • 使用後は必ずDispose()メソッドでリソースを解放
  • Commonクラスの統一されたエラー処理システムを活用
  • エラーコードを適切に使用して問題の特定を効率化
  • ログを活用してデバッグを効率化

🚨 エラーコード一覧

各ドライバークラスには固有のエラーコードが設定されています。エラーコードを確認することで、問題の原因を特定しやすくなります。

エラーコード範囲

1000番台 WebDriver関連
2000番台 EdgeDriver関連
3000番台 ChromeDriver関連
4000番台 WordDriver関連
5000番台 ExcelDriver関連
6000番台 PowerPointDriver関連
7000番台 OracleDriver関連
9000番台 Common関連（カスタムスクリプト用）

🔧 エラー対処のヒント

  1. エラーコードを確認して原因を特定
  2. AllDrivers_Error.logファイルで詳細を確認
  3. 必要に応じて再試行ロジックを実装
  4. リソースの解放を確実に実行
